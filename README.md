**Temperature Indicator PCB**

**📌 Project Overview**

This project demonstrates a simple temperature indicator circuit designed using KiCad. The circuit uses a NTC Thermistor to detect temperature changes and a BC547 to control an LED indicator.
The LED turns ON when the temperature increases, providing a simple visual indication of temperature change.

**⚙️ Features**

 Simple temperature sensing circuit

**🧩 Components Used**

 NTC Thermistor (10kΩ)
 
 BC547
 
 LED
 
 Resistor (10kΩ)
 
 Resistor (220Ω)

2-pin power connector (5V supply)

**🔌 Working Principle**

 The NTC thermistor changes its resistance depending on temperature.
 
 When the temperature increases, the resistance of the thermistor decreases.
 
 This increases the voltage applied to the base of the BC547 transistor.
 
 The transistor turns ON, allowing current to flow from collector to emitter.
 
 Current passes through the LED and resistor, causing the LED to glow.
 
 Thus, the LED acts as a temperature indicator.
